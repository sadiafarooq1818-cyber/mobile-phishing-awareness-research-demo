# Project 01: Retail Sales Excel Dashboard

## Business Objective

A medium-sized retail business needs a clear reporting pack to monitor sales performance, category performance, regional performance and monthly trends. This project demonstrates how raw sales data can be organised into dashboard-ready summaries for management reporting.

## Key Business Questions

- What is total sales revenue and profit by month?
- Which product categories generate the highest revenue?
- Which regions are performing above or below target?
- What are the top-selling products?
- Where are the largest discounts being applied?

## Dataset

Sample sales data is provided in `data/retail_sales_sample.csv`.

Key fields include:

- Order ID
- Order Date
- Region
- State
- Customer Segment
- Product Category
- Product Name
- Sales
- Quantity
- Discount
- Profit

## Excel Dashboard Features Demonstrated

- Data cleaning and formatting
- PivotTable-ready structure
- KPI cards for revenue, profit, quantity and margin
- Monthly trend analysis
- Region and category performance summaries
- Slicer-ready fields for region, category and customer segment
- Conditional formatting for performance review
- Dashboard documentation for repeatable reporting

## Recommended Dashboard Pages

1. Executive Overview
2. Sales Trend Analysis
3. Regional Performance
4. Product Category Performance
5. Customer Segment Analysis

## Core KPIs
