# Data Analytics & Business Intelligence Portfolio

This folder contains demonstration projects designed to show practical skills for Data Analyst, Reporting Analyst and Business Intelligence roles.

## Portfolio Projects

1. **Retail Sales Excel Dashboard**  
   Demonstrates Excel dashboard planning, KPI reporting, PivotTable-style summaries, slicer-ready fields, data validation and stakeholder reporting.

2. **Power BI Retail Performance Dashboard**  
   Demonstrates BI dashboard design, star-schema thinking, DAX measures, KPI cards, trend analysis, category analysis and dashboard documentation.

3. **SQL Customer & Revenue Analysis**  
   Demonstrates SQL querying, joins, grouping, filtering, customer segmentation, revenue analysis and business-question-based reporting.

4. **Python Data Cleaning & Insights Demo**  
   Demonstrates Python data cleaning, missing-value handling, duplicate checks, quality validation, feature preparation and insight generation.

## Skills Demonstrated

- Data cleaning and validation
- SQL querying and business analysis
- Excel reporting and dashboard planning
- Power BI dashboard design and DAX measures
- Python analysis using pandas
- KPI reporting and performance monitoring
- Data documentation and business communication
- Dashboard-ready dataset preparation

## Note

The datasets included here are sample demonstration datasets created for portfolio purposes. They are designed to show analytical thinking, reporting structure and business intelligence workflow capability.
