# Project 04: Python Data Cleaning & Insights Demo

## Business Objective

This project demonstrates how Python can be used to clean, validate and analyse customer support data. It reflects common tasks in Data Analyst and Reporting Analyst roles, including missing-value handling, duplicate detection, data type formatting, KPI creation and summary reporting.

## Business Questions

- How many support tickets were received by category?
- What is the average resolution time?
- Which ticket priority has the longest resolution time?
- Which channels generate the highest volume of tickets?
- Are there missing values or duplicate records that need to be cleaned?
- What insights should be shared with business stakeholders?

## Dataset

Sample data is provided in:

`data/customer_support_tickets.csv`

## Python Skills Demonstrated

- pandas data loading
- data quality checks
- missing-value handling
- duplicate detection
- date parsing
- KPI calculation
- grouped summaries
- data export for reporting
- stakeholder-friendly insight writing

## Files

- `scripts/data_cleaning_insights.py` contains the Python analysis workflow.
- `data/customer_support_tickets.csv` contains the sample dataset.
