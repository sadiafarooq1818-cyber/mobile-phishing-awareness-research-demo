# Project 03: SQL Customer & Revenue Analysis

## Business Objective

This project demonstrates how SQL can be used to answer business questions for a medium-sized retail or service organisation. The analysis focuses on customer segments, revenue, product categories, order patterns and regional performance.

## Business Questions

- What is total revenue by region?
- Which customer segment generates the highest revenue?
- Which product categories have the strongest profit contribution?
- What are the top 10 products by sales?
- Which orders should be reviewed due to high discounting or low margin?
- What monthly sales trend is visible in the dataset?

## Dataset

Use the retail dataset from:

`../project-01-retail-sales-excel-dashboard/data/retail_sales_sample.csv`

## SQL Skills Demonstrated

- SELECT statements
- WHERE filtering
- GROUP BY aggregation
- ORDER BY ranking
- CASE statements
- Common Table Expressions concept
- Business-question-based query writing
- KPI calculation
- Data quality checks
- Revenue and margin analysis

This project is relevant for Data Analyst, BI Analyst, Reporting Analyst and Business Analyst roles because it demonstrates the ability to convert business questions into structured SQL analysis.
